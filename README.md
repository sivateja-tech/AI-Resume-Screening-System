# AI Resume Screening System

An AI-powered web application that analyzes resumes and detects candidate skills using NLP techniques.

## Features
- Upload Resume PDF
- Extract Resume Text
- Detect Technical Skills
- Simple Web Interface
- Resume Analysis using Python

## Technologies Used
- Python
- Flask
- Machine Learning
- HTML/CSS
- PyPDF2

## Project Structure

```bash
resume-screening-system/
│
├── app.py
├── model.py
├── requirements.txt
├── templates/
├── static/
└── uploads/
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```

## Output
The system analyzes uploaded resumes and displays detected skills based on predefined categories.
